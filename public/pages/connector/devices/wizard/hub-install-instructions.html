Our Hub runs on Mac, Linux, and Windows. It's system requirements are small enough to run on a Raspberry Pi. Here are the instructions on how to install your Hub:

1. Verify that Node.JS is installed on your machine. You can do this by opening a command line window or terminal and typing: 

node -v

If your system returns a version number and it's greater than 8.0 please proceed to step 2; otherwise, goto to nodejs.org and download/install Node.JS following their instructions - and then proceed to step 2.

2. Download and unzip our Hub from GitHub. Goto your new unzipped directory and run the following commands: 

npm install
node server.js

3. Select 'Next' to start hacking!
